#Chatty App
=====================

A chat app, using Websockets and ReactJS

### Usage

Clone the repo

```
git clone https://github.com/JohnTheScout/chatty-app.git
cd chatty-app
```

Install the dependencies and start the server.

```
npm install
npm start
open http://localhost:3000
```
